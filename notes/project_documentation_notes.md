## Project Goal

Create an LC-3 Assembly program that accepts five test scores and calculates:

- Minimum score
- Maximum score
- Average score
- Letter grade

## Program Flow

1. Read five scores from the user.
2. Validate each score.
3. Initialize minimum, maximum, and sum values.
4. Process remaining scores.
5. Update minimum score if necessary.
6. Update maximum score if necessary.
7. Add score to running total.
8. Calculate average score.
9. Determine letter grade.
10. Display results.

## Planned Subroutines

- READ_SCORES
- INPUT_VALIDATION
- UPDATE_MIN
- UPDATE_MAX
- ADD_TO_SUM
- CALCULATE_AVERAGE
- FIND_LETTER_GRADE
- DISPLAY_RESULTS
