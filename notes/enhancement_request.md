# Enhancement Requests

## Version 2 Ideas

### Variable Number of Scores

Allow the user to enter any number of test scores instead of exactly five.

Benefits:
- More flexible
- Better real-world usability

Potential implementation:
- Dynamic memory allocation
- Variable loop counter
- Average = Sum / NumberOfScores

---

### Improved Input Validation

Reject invalid scores and prompt the user to re-enter them.

Examples:
- Negative scores
- Scores greater than 100

### Additional Statistics
