# CIS-11 Test Score Calculator

Team R7

Members:
- Jullian Toledo
- Rishik Reddy Kasarla
- Brendyn Keaton

Project Description:
LC-3 Assembly program that accepts 5 test scores and calculates:

- Minimum score
- Maximum score
- Average score
- Letter grade

Features:
- Accepts 5 scores
- Input Validation (0-100)
- Error handling and re-prompting
- ASCII conversion
- PUSH-POP operation on stack
- Min/Max calculation
- Average calculation
- Letter grade determination
- Display all values
- Modular design using subroutines

Files:
- Test Calculators_Brendyn Keaton.asm (Final submission version)
- test_calculator.asm
- test_calculator_rishik.asm
- test_score_calculator_workingV1.asm

Testing:
- The program was tested using valid and invalid inputs.

Example:
Input:
52
87
96
79
61

Output:
Minimum Score: 52
Maximum Score: 96
Average score: 75
Letter Grade: C
